半反应

编辑

半反应是氧化还原反应中氧化或还原反应部分。半反应通常是考虑氧化还原反应中所涉及的单个物质氧化态的变化。[1]

通常，半反应的概念被用来描述电化学电池中发生的事情，例如原电池。可以通过书写半反应来描述经历氧化的金属(称为阳极)和经历还原的金属(称为阴极)。

半反应通常被用作配平氧化还原反应的方法。对于酸性条件下的氧化还原反应，在配平原子和氧化数后，需要加入H+来平衡半反应中的氢离子。对于碱性条件下的氧化还原反应，在配平原子和氧化数后，首先将其作为酸性溶液处理，然后加入OH-以平衡半反应中的H+(这会产生H2O)。

1 例子:锌和铜原电池编辑

伽伐尼电池

考虑下图中显示的原电池:它由一块浸在硫酸锌 (ZnSO4)溶液中的锌 (Zn)和一块浸在硫酸铜(CuSO4)溶液中的铜(Cu)构成。总反应是:

Zn(s) + CuSO4(aq) → ZnSO4(aq) + Cu(s)

锌阳极发生了氧化(金属失去电子)。这表现在以下氧化半反应中(注意电子在产物侧):

Zn(s) → Zn2+ + 2e−

铜阴极发生了还原(金属得到电子)。这表现在以下还原半反应中(注意电子在反应物一侧):

Cu2+ + 2e− → Cu(s)

2 例子:镁的氧化编辑

合成碱性氧化物的实验。镁带被燃烧器点燃。镁燃烧发出强光，形成氧化镁（MgO）。

一张燃烧的镁带的照片，曝光时间很短，可以获得氧化细节。

考虑镁带（Mg）燃烧的例子。当镁燃烧时，它与空气中的氧(O2)结合，根据以下方程形成氧化镁(MgO):

2Mg(s) + O2(g) → 2MgO(s)

氧化镁是一种含有Mg2+和O2-的离子化合物，而Mg(s)和O2(g)是不带电荷的元素。零电荷的镁从反应物侧到产物侧获得+2电荷，零电荷的O2(g)获得-2电荷。这是因为当Mg(s)变成Mg2+时，它失去了2个电子。由于左侧有2个镁（Mg)，根据以下氧化半反应，总共损失了4个电子:

2Mg→2Mg 2++ 4e-

另一方面，O2被还原:其氧化态从0变为-2。因此，当O2获得4个电子时，可以将O2写入还原半反应:

O2(g)+4e-→2O 2-

总反应是两个半反应的总和:

2Mg(s) + O2(g) + 4e− →2Mg2+ + 2O2− + 4e−

当化学反应特别是氧化还原反应发生时，我们看不到电子在反应过程中的出现和消失。我们看到的是反应物(原料)和最终产物。因此，出现在等式两边的电子被抵消了。相消除后，等式被重写为

2Mg(s) + O2(g) →2Mg2+ + 2O2−

两种离子，正(Mg2+)和负(O2-)存在于产品侧，由于它们具有相反的电荷(静电引力)，二者立即结合形成化合物氧化镁(MgO)。在任何给定的氧化还原反应中，都有两个半反应——氧化半反应和还原半反应。这两个半反应的总和就是氧化还原反应。

3 半反应配平法编辑

考虑以下反应:

Cl2 + 2Fe2+ → 2Cl− + 2Fe3+

铁和氯这两种元素各自改变氧化状态；铁从+2到+3，氯从0到-1。这里然后有两个有效的半反应发生。这些变化可以通过在每个半反应中加入适当的电子来用反应式表示:

Fe2+ → Fe3+ + e−

Cl2 + 2e− → 2Cl−

给定两个半反应，在知道适当电极电位的情况下，有可能以同样的方式达到完全(原始)反应。将反应分解成半反应是理解各种化学过程的关键。例如在上述反应中，可以表明这是氧化还原反应，其中铁被氧化，氯被还原。注意电子从铁转移到氯。分解也是简化化学方程式配平的一种方法。化学家可以一次得到一个原子配平和电荷配平的方程式。

例如:

  • Fe2+ → Fe3+ + e− 变成2Fe2+ → 2Fe3+ + 2e−
  • 添加到Cl2 + 2e− → 2Cl−
  • 最后变成Cl2 + 2Fe2+ → 2Cl− + 2Fe3+

在碱性或酸性条件下考虑半反应也是可能的，有时也是必要的，因为在氧化还原反应中可能存在酸性或碱性电解质。由于这种电解质，可能更难满足原子和电荷的平衡。这是通过在反应的任一侧添加OH−， H2O，e−或H+来实现的，直到原子和电荷平衡。

考虑下面的半反应:

PbO2 → PbO

OH−， H2O和e−  可以用来平衡碱性条件下的电荷和原子，只要假设反应在水中。

2e− + H2O + PbO2 → PbO + 2OH−

再次考虑下面的半反应:

PbO2 → PbO

OH−， H2O和e− 可以用来平衡酸性条件下的电荷和原子，只要假设反应在水中。

2e− + H2O + PbO2 → PbO + 2OH−

请注意，两边都是电荷平衡和原子平衡的。

在酸性和碱性条件下，通常会同时存在 H+和OH−  ，但这两种离子的反应会产生水H2O(如下所示):

H+ + OH− → H2O
