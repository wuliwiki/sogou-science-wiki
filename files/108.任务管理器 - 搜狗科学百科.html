任务管理器

编辑

任务管理器，以前称为Windows任务管理器，集任务管理器，系统监视器和启动管理器为一体，是微软Windows系统的一部分。它提供有关计算机性能和运行软件的信息，包括运行进程的名称、CPU负载、已提交内存、输入/输出详细信息、登录用户和Windows服务。任务管理器还可用于设置进程优先级、处理器关联性、启动和停止服务以及强制终止进程。

在近期版本的Windows中，通过按⊞ Win+R，然后键入taskmgr.exe，或者按Ctrl+Alt+Delete，然后单击开始任务管理器，或是按Ctrl+⇧ Shift+Esc，或右键单击Windows任务栏并选择“任务管理器”，可以启动该程序。

任务管理器以当前的形式在Windows NT 4.0中引入。早期版本的Windows NT，以及Windows 3.x，包括任务列表应用程序，能够列出当前运行的进程并结束它们，或者创建一个新的进程。Windows 9x有一个名为“关闭程序”的程序，它可以列出当前正在运行的程序，并提供关闭程序和关闭计算机的选项。[1]

1 功能编辑

Windows XP上的任务处理器显示的进程选项卡

任务管理器有两种视图。用户第一次调用任务管理器时，它以简化的摘要模式显示(在用户体验中描述为“简略信息”)。通过单击“详细信息”，可以切换到更详细的模式。这台机器会为用户记住此设置。[2]

1.1 摘要模式

在摘要模式下，任务管理器显示当前运行的有主窗口的程序列表。它有一个“详细信息”超链接，点击可以激活完整的包含多个选项卡任务管理器。

右键单击列表中的任何应用程序都可以切换到该应用程序或结束应用程序。发出“结束任务”会向应用程序发送从容退出请求。

1.2 进程和细节

“进程”选项卡显示系统上所有正在运行的进程的列表。该列表包括来自其他帐户的Windows服务和进程。删除键也可用于终止进程选项卡上的进程。默认情况下，“进程”选项卡显示运行该进程的用户帐户、CPU数量以及该进程当前消耗的内存量。可以显示更多的列。“进程”选项卡将流程分为三类：

  • 应用程序：带有主窗口的程序
  • Windows进程：没有主窗口的Windows本身的组件，包括服务
  • 后台进程：没有主窗口的程序，包括服务，并且不是Windows本身的一部分

该选项卡显示每个主窗口的名称以及与每个进程相关联的服务的名称。根据命令是发送到进程还是该进程的窗口，可以从该选项卡发送正常退出命令和终止命令。

“详细信息”选项卡是“进程”选项卡的更基础版本，其作用类似于Windows 7和更早版本中的“进程”选项卡。它具有更基本的用户体验，并且可以执行一些额外的操作。右键单击列表中的某个进程可以更改该进程的优先级，设置处理器关联性(设置该进程可以在哪个或哪些CPU上执行)，并可以结束该进程。选择“结束进程”会导致Windows立即终止进程。选择“结束进程树”会导致Windows立即终止该进程，以及由该进程直接或间接启动的所有进程。与从“应用程序”选项卡中选择“结束任务”不同，当选择“结束进程”时，程序不会得到警告，也没有机会在结束前进行清理。但是，当一个进程运行的安全上下文不同于发出终止进程调用的进程时，需要使用KILL命令行终止程序。[3]

1.3 性能

“性能”选项卡显示系统性能的总体统计数据，最显著的是总CPU使用量和正使用的内存量。该选项卡显示了这两个值的最近使用情况图表。还显示了特定内存区域的详细信息。

有一个选项可以将CPU使用图分成两部分：内核模式时间和用户模式时间。许多设备驱动程序和操作系统的核心部分以内核模式运行，而用户应用程序以用户模式运行。可以通过从“视图”菜单中选择“显示内核时间”来打开该选项。当该选项打开时，CPU使用率图表将显示绿色和红色区域。红色区域表示在内核模式下花费的时间，绿色区域表示在用户模式下花费的时间。

“性能”选项卡还显示与计算机中存在的每个网络适配器相关的统计信息。默认情况下，会显示适配器名称、网络利用率百分比、链路速度、网络适配器的状态以及最近活动的图表。

1.4 应用历史

“应用程序历史记录”选项卡显示Universal Windows Platform应用程序的资源使用信息。Windows对这些应用程序的生命周期的控制更严格。此选项卡可以查看Windows收集的关于它们的数据。

1.5 启动

“启动”选项卡管理以Windows shell开头的软件。

1.6 用户

“用户”选项卡显示当前在计算机上有会话的所有用户。在服务器计算机上，可能有多个用户使用终端服务连接到计算机。从Windows XP开始，也可能有多个用户同时使用快速用户切换功能登入电脑。用户可以从该选项卡断开连接或注销。

2 历史编辑

2.1 Windows 9x

当在Windows 9x中按下Ctrl+Alt+Delete时，会出现一个“关闭程序”对话框。[1] 此外，在Windows 9x中，有一个位于Windows目录中的名为Tasks (TASKMAN.EXE) 的程序。TASKMAN.EXE较为基础，特征较少。Windows 9x中的系统监视器实用程序包含类似于Windows任务管理器的进程和网络监视功能。(此外，如果浏览器进程关闭，通过在桌面上单击两次来调用Tasks程序。)

2.2 Windows XP

仅在Windows XP中存在关机菜单，提供对待机、休眠、关机、重启、注销和切换用户的访问。更高版本的Windows通过开始菜单提供这些选项。

在“性能”选项卡上，CPU值的显示从LED七段显示的显示更改为标准数值。这样做是为了适应非阿拉伯数字系统，例如东方阿拉伯数字，不能用七段显示来表示。[4]

在Windows XP之前，长度超过15个字元的程序名称会被截断。这个问题在Windows XP系统中得到解决。[5]

用户选项卡是由Windows XP系统引入的。

从Windows XP开始，“删除”键在“进程”选项卡上启用。

2.3 Windows Vista

Windows任务管理器已在Windows Vista中更新了新功能，[6]包括:

  • “服务”选项卡，用于查看和修改当前运行的Windows服务，启动和停止任何服务，以及启用/禁用用户账户控制 (UAC) 文件和进程的注册表虚拟化。
  • “进程”选项卡中新的“图像路径名”和“命令行”以及“描述”栏。这些显示了进程中运行的可执行映像的全名和路径、提供的任何命令行参数以及映像文件的“描述”属性。
  • 显示DEP状态和虚拟化状态的新栏。虚拟化状态指的是UAC虚拟化，在该状态下，对某些系统位置的文件和注册表引用将被静默方式重定向到用户特定的区域。
  • 通过右键单击任何进程，可以直接打开该进程的可执行映像文件的属性或包含该进程的目录（文件夹）的属性。
  • 任务管理器也不再容易受到来自远程源或病毒的攻击，因为它必须在管理权限下运行才能执行某些任务，例如注销其他连接的用户或发送消息。用户必须进入“进程”选项卡并单击“显示来自其他用户的进程”，以验证管理权限并解锁这些权限。除非禁用UAC，否则显示所有用户的进程要求包括管理员在内的所有用户接受UAC提示。若用户不是管理员，则必须在提示继续时输入管理员账户的密码，但在禁用UAC时，将不会提升权限的程序。
  • 通过右键单击任何正在运行的进程，可以创建转储。如果应用程序或进程没有响应，此功能会很有用，这样可以在调试器中打开转储文件以获取更多信息。
  • 包含待机、休眠、关闭、重启、注销和切换用户的关机菜单已被删除。这样做是因为该菜单使用率低，并且删除后降低了任务管理器的整体复杂性。[7]
  • “性能”选项卡显示系统正常运行时间。

2.4 Windows 8

Windows8中taskmgri.exe的不同部分的图的集合 (右下角是图文摘要)

在Windows 8中，Windows任务管理器已全面调整，并作了以下更改：[2]

  • 从Windows 8开始，默认情况下选项卡是隐藏的，任务管理器以摘要模式打开。此视图仅显示应用程序及其相关流程。在Windows 8之前，摘要模式中显示的内容显示在名为“应用程序”的标签中。
  • “进程”选项卡中的资源利用率显示为不同深浅度的黄色，较深的颜色表示深度使用。
  • 性能选项卡分为CPU、内存、磁盘、以太网和无线网络 (如果适用) 部分。每个都有总体图表，点击其中一个可以得到特定资源的详细信息。它整合了从Windows XP到Windows 7出现在“网络”选项卡中显示的信息。
    • 默认情况下，“CPU”选项卡不再为系统上的每个逻辑处理器显示单独的图表。现在它可以显示每个NUMA节点的数据。
    • “CPU”选项卡现在在热映射图上显示简单的百分比，以显示具有多个 (64个或更多，最多640个) 逻辑处理器的系统的利用率。[8] 用于这些热映射图的颜色是蓝色，较暗的颜色再次表明深度使用。
    • 将光标悬停在任何逻辑处理器的数据上，现在会显示该处理器的NUMA节点及其标识。
  • 添加了一个新的“启动”选项卡，列出了正在运行的启动应用程序。[9]
  • “进程”选项卡现在列出了每个进程的应用程序名称、应用程序状态以及CPU、内存、硬盘和网络资源的总体使用数据。
  • 引入了一个新的“应用历史”选项卡。
  • 引入了“启动”选项卡。以前，MSConfig负责这项任务，而在Windows Vista中，Windows Defender的“应用和浏览器控制”部分负责这项任务。[10] (内置于Windows7中的Windows Defender没有这个选项，也没有出现在可下载的Microsoft Security Essentials中)
    • 应用程序状态可以更改为挂起。
    • 旧版任务管理器中的常规过程信息可以在新的“详细信息”选项卡中找到。

2.5 Windows 10

“进程”选项卡分为多个类别。[11]

3 缺点编辑

任务管理器是计算机病毒和其他形式恶意软件的常见目标；通常，恶意软件会在任务管理器启动后立即将其关闭，以便对用户隐藏自己。例如，Zotob和Spybot蠕虫的变种已经使用了这种技术。[12] 使用组策略，可以禁用任务管理器。许多类型的恶意软件也会在注册表中启用此策略设置。Rootkits可以防止自己被列在任务管理器中，从而防止用户使用任务管理器检测和终止它们。
